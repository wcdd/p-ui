# p-ui · SOCKS5 (S5) 代理一键管理脚本

![Shell](https://img.shields.io/badge/shell-bash-1f425f)
![Platform](https://img.shields.io/badge/platform-Debian%20|%20Ubuntu%20|%20CentOS-blue)
![Proxy](https://img.shields.io/badge/proxy-Dante%20SOCKS5-green)
![License](https://img.shields.io/badge/license-MIT-blue)

一个基于 **Dante** 的 SOCKS5 代理一键管理脚本。一条命令装好依赖、随机生成账号端口、菜单化管理，并支持 **IP 白名单（只允许指定 IP 访问）**。

> 输入 `p-ui` 即可随时打开管理菜单。

---

## ✨ 功能特性

- **一键安装**：自动安装 SOCKS5 所需依赖并完成配置（Debian/Ubuntu 用 apt；CentOS/RHEL 自动编译）
- **随机生成 S5**：自动生成随机端口 + 随机用户名 + 随机密码
- **IP 白名单**：可设置只允许指定 IP / 网段访问，其余一律拒绝
- **两种认证模式**：账号密码认证，或「仅 IP 白名单（免密码）」，一键切换
- **菜单化管理**：改端口、换账号、启停服务、查看连接信息、卸载，全部菜单操作
- **快捷命令**：安装后在任意位置输入 `p-ui` 即可打开菜单
- **防开放代理保护**：自动拦截「免密码 + 无白名单」这类危险组合
- **自动放行防火墙**：自动配置系统内的 ufw / firewalld 端口

---

## 📦 系统要求

- **Debian 9+ / Ubuntu 18.04+**（推荐，apt 直装，最快）
- **CentOS 7+ / RHEL / Rocky / AlmaLinux**（仓库无 Dante，脚本会自动下载源码编译，约 1–3 分钟）
- 需要 **root** 权限
- 一台具有公网 IP 的 VPS

```bash

```

## 📖 使用方法

首次运行后，脚本会把自己安装到 `/usr/local/bin/`，并创建快捷命令 `p-ui`。之后随时输入 `p-ui` 即可打开下面的菜单：

| 选项 | 功能 |
| :--: | :--- |
| `1` | 安装 / 重装 S5 代理（含一键装依赖） |
| `2` | 查看 S5 配置信息（地址、端口、账号、连接串） |
| `3` | 随机重新生成账号密码 |
| `4` | 修改端口 |
| `5` | 切换认证方式（账号密码 ⇄ 仅 IP 白名单） |
| `6` | 查看白名单 |
| `7` | 添加白名单 IP |
| `8` | 删除白名单 IP |
| `9` | 清空白名单（允许所有 IP） |
| `10`–`13` | 启动 / 停止 / 重启 / 查看服务状态 |
| `14` | 卸载 S5 代理 |
| `15` | 安装 / 更新 p-ui 快捷命令 |
| `0` | 退出 |

安装完成后，菜单会给出两种格式的连接信息，方便直接复制到客户端：

```
连接串   : socks5://用户名:密码@服务器IP:端口
通用格式 : 服务器IP:端口:用户名:密码
```

## 🔒 IP 白名单（只允许指定 IP 访问）

这是本脚本的核心功能之一，用来限制谁能连你的代理。

- **添加白名单**：菜单 `7`，支持单个 IP（如 `1.2.3.4`）或网段（如 `1.2.3.0/24`）。
- **一旦设置了白名单**，就只有名单内的 IP 能连接，其余全部拒绝。
- **白名单为空时**（默认）：允许任意 IP 连接，但仍需账号密码。

### 两种认证模式（菜单 `5` 切换）

| 模式 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **账号密码**（默认） | 任意 IP 可连，但必须输入用户名密码 | 你的出口 IP 经常变（如家庭宽带） |
| **仅 IP 白名单** | 名单内 IP 免密码直接用，名单外拒绝 | 你的出口 IP 固定（如公司/另一台服务器） |

> ⚠️ 切换到「仅 IP 白名单」模式前**必须先添加至少一个白名单 IP**，否则会变成任何人都能用的**开放代理**。脚本会自动拦截这种危险组合。

## 🔌 客户端连接与测试

安装后可用 `curl` 快速验证代理是否生效（返回结果应为你的**服务器公网 IP**）：

```bash
# 账号密码模式
curl -x socks5h://用户名:密码@服务器IP:端口 https://api.ipify.org

# 仅 IP 白名单模式（免密码）
curl -x socks5h://服务器IP:端口 https://api.ipify.org
```

常见客户端填写方式：代理类型选 **SOCKS5**，填入服务器 IP、端口，需要认证则填用户名和密码。

## ❓ 常见问题 / 故障排查

**Q：本地能 ping 通但代理连不上？**
A：最常见原因是**云服务商的安全组 / 防火墙**没放行端口。脚本只能配置系统内的 ufw / firewalld，配不到阿里云、腾讯云、AWS、Vultr 等控制台里的「安全组」，请去控制台手动放行对应端口（TCP）。

**Q：服务启动失败怎么排查？**
A：查看日志：

```bash
journalctl -u danted -n 30 --no-pager
```

**Q：CentOS 上安装很慢？**
A：因为 CentOS/RHEL 官方源里没有 Dante，脚本会自动下载源码编译，需要 1–3 分钟，属正常现象。若编译下载地址失效，可修改脚本中 `compile_dante` 函数里的版本号。

**Q：一键命令用管道（`bash <(curl ...)`）跑，`p-ui` 命令没装上？**
A：脚本通过复制自身来安装快捷命令，管道运行时拿不到脚本文件本体。请改用文档里「先下载到文件再运行」的方式，`p-ui` 命令即可自动安装。

## 🗑️ 卸载

打开菜单选择 `14`，会停止服务、删除配置与账号；并可选择一并删除 `p-ui` 命令与脚本。

---

## 🛡️ 安全建议

- 优先使用**账号密码 + IP 白名单**双重限制。
- 不要把端口设成常见端口（如 1080），脚本默认会随机生成。
- 定期用菜单 `3` 更换账号密码。
- 账号密码信息保存在 `/etc/danted.s5info`（仅 root 可读）。

## 🧩 工作原理 / 文件位置

底层使用成熟开源的 **Dante (danted)** 作为 SOCKS5 服务端。相关文件：

| 路径 | 用途 |
| :--- | :--- |
| `/etc/danted.conf` | Dante 主配置（由脚本自动生成，勿手改） |
| `/etc/danted.s5info` | 当前账号 / 端口信息（仅 root 可读） |
| `/etc/danted.whitelist` | IP 白名单（每行一个） |
| `/etc/systemd/system/danted.service` | systemd 服务 |
| `/usr/local/bin/p-ui` | 快捷命令 |
